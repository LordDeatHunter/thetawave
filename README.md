# Thetawave

## Controls

- Movement: WASD or Arrow Keys
- Fire Weapon: Left Click (can hold) or Spacebar

## Description

Thetawave is an open-source game, designed to be easily expandable. It is programmed in the Rust Language using the Bevy Engine.
